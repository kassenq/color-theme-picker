# Color Theme Picker

A React-based color theme picker application. Try [here](https://kassenq.github.io/color-theme-picker/).

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Features

- Interactive color picker
- Real-time color preview
- Hex color code display

## Tech Stack

- React 18
- Vite
- CSS3
